Feedback Widget Test Page

Instructions:

  1. Look for a floating button on the right side of the window
  2. It should say "Feedback" and be rotated vertically
  3. Click it to open the feedback modal
  4. Fill out the form and submit

Expected Behavior:

  • Button appears on right side, centered vertically
  • Button has ISRS gradient colors (blue to teal)
  • Hovering reverses the gradient
  • Clicking opens a modal dialog
  • Form includes: name, email, component, type, rating, message
  • Submitting shows success message

Debug Console:

Open browser DevTools (F12) and check Console for any errors
